a deployment risk simulator. The aesthetic must feel like a Super Nintendo or 
Sega Genesis era game HUD crossed with a beach sandbox toy world. Use the logo named sandbox-log.png

Visual style:
- Warm sandy color palette: golden sand (#e8c96a), dark brown wood (#6b4226), 
  bright sand highlight (#f5e6a3), deep shadow brown (#2e1a0e)
- Accent colors: ocean blue (#3db9f5), flag red (#e8402a), shovel blue (#4a90d9)
- Dark near-black warm background (#1a0f02) like a night beach
- All UI elements must look like pixel sprites — chunky borders, 3D bevel 
  box shadows, pixel chamfer corners (not rounded, cut diagonally)
- CRT scanline overlay effect on panels (subtle horizontal lines)
- Pixel dithering on gradients instead of smooth blends

Typography:
- Headers: "Press Start 2P" Google Font — sandy golden color with 3px dark 
  brown drop shadow offset to bottom-right (like the Sandbox logo lettering)
- Data/terminal text: "VT323" Google Font — tall, retro terminal readout style
- Body: "Rajdhani" — clean military stencil feel

Layout — 3 panel game HUD:
- LEFT PANEL: System Map — pixel node graph with glowing dotted connection 
  lines, nodes styled like 16-bit game icons
- CENTER PANEL: "Future Failure Replay" — a vertical timeline styled like 
  an RPG battle log, events appear one by one with typewriter effect
- RIGHT PANEL: Risk Report — styled like a game stats screen, big chunky 
  score number, HP-style bar that drains red as risk increases

Key UI components:
- Buttons: raised pixel bevel (inset box shadows top-left light, 
  bottom-right dark), sand-colored like physical wooden toy blocks
- Panels: dark wood-grain bordered frames like a sandbox wooden border
- Top HUD bar: sticky header like a SNES game status bar — shows project 
  name, live status dot, risk badge
- Risk badge: pixel chamfer corners, glows and pulses red on CRITICAL
- Drop zone upload area: giant centered zone with pixel corner brackets, 
  animated scanline sweep, crate/box pixel art icon in the center
- Sand dune silhouette along the bottom of the upload screen (jagged 
  pixel art terrain)
- Small decorative pixel objects: sandcastle, beach ball, toy shovel, 
  red flag planted in sand

Animations:
- Scanline sweep (vertical light beam moving top to bottom on panels)
- Pixel particles burst when file is dropped
- Crate lid flips open on drag-over
- Timeline events slide in from bottom one at a time like an RPG log
- Risk score counts up from 0 with a tick sound feel
- Nodes on system map pulse gently and send animated dots along connection lines

The overall feel: you are playing a 1994 SNES strategy game but the game 
world IS your software system. Dropping a ZIP feels like dropping a 
cartridge into a console. The failure replay feels like watching your
